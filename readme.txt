== Description ==

Copytrack detects where your images has been used on the web and assist you in the legal process, for free. This plugin will upload your Media Library to Copytrack.

=== USAGE ===
Create an account at Copytrack (it is free). Visit the Meow Apps > Copytrack. Follow the instructions. Go in the Media Library, and start uploading your files. Make sure you wait 2-3 seconds between the end of the upload and a new one, otherwise the request might be rejected by Copytrack. Bulk upload is also available, for that visit the Meow Apps > Copytrack menu in your WordPress Admin.

=== 3RD PARTY SERVICE ===
Copytrack is a third-party service and this plugin is not affiliated with it. The Image Copytrack plugin is accessing the Copytrack API using your key and token. It is totally safe, however, I cannot be hold responsible for how your images and data is used by [Copytrack](https://copytrack.com). Please check their [Terms & Conditions](https://www.copytrack.com/copytrack-terms-conditions/) and their [Private Policy](https://www.copytrack.com/privacy/) to know more.

=== CLEAN ===
Nothing is created in the database except a tiny meta entry for that media, specific to Copytrack.

Languages: English.

== Installation ==

1. Upload `image-copytrack` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Upgrade Notice ==

Replace all the files. Nothing else to do.
